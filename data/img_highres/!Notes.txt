Notes
-----

All box art images end with the suffix "-1.iff"
All title screens end with the suffix "-2.iff"
The first in-game screen shot ends with the suffix "-3.iff"

Some games might not have box art or title screen. Therefore, sometimes the
first image may end in "-2.iff" or even "-3.iff".

Additionally, some games don't have less than 5 in-game screenshots, but
these will always start with the "-3.iff" suffix.

CD32 games have been excluded because there's no 100% reliable way to link
the data together between the two platforms (Amiga and CD32) in the OpenRetro
database.

Because of the way in which the OpenRetro database has a parent record per
chipset, it does not provide separate screenshots for different languages,
which means it has the same English images for all versions.
